# Chronicle v1.0.0 Release Draft

## Tag

`v1.0.0`

## Release Title

`Chronicle 1.0.0`

## Release Body

Chronicle 1.0.0 is the first public release of the current standalone Chronicle app line.

This release replaces the outdated legacy public repository snapshot with the rebuilt Chronicle desktop app and its current documentation set.

### Highlights

- accessibility-first GUI designed for real keyboard and screen-reader workflows
- support for difficult scans, reports, archival material, handwriting, newspapers, books, and mixed-format batches
- multi-provider support across Gemini, Claude, and OpenAI
- queue controls, session recovery, and safer review-oriented workflows
- first packaged public builds for macOS and Windows

### Release Assets

- `Chronicle.1.0.mac.zip`
- `Chronicle.1.0.windows.zip`

### Important Notes

- Chronicle is designed to reduce cleanup and improve reviewability on difficult documents.
- Human review is still required, especially for legal, medical, financial, safety-critical, and compliance-sensitive material.
- Users provide their own API keys on their own machines.
- Document content is processed by the selected provider API.
