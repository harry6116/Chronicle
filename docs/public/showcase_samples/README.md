# Chronicle Showcase Samples

These HTML files are public-safe Chronicle output samples used by the website showcase and public repo demo material.

## Included samples

- `varus-velleius-sample.html`
  - Chronicle output from Velleius Paterculus on the Varus disaster.
  - Source edition: <https://archive.org/details/bim_eighteenth-century_the-roman-history-of-c-_velleius-paterculus_1724>

- `varus-tacitus-sample.html`
  - Chronicle output from Tacitus on Germanicus revisiting the Varus battlefield.
  - Source edition: <https://archive.org/details/historicalannals01taci>

- `cromwell-letters-sample.html`
  - Chronicle output from an Oliver Cromwell letters volume.
  - Source edition: <https://archive.org/details/olivercromwells01cromiala>

## Historical note

The exact original dispatch to Augustus reporting the loss of the three Roman legions does not survive. The Varus showcase therefore uses the closest surviving ancient-source witnesses staged for public demonstration.
